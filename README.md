# School Canteen

Веб-приложение для автоматизации учета питания в школьной столовой.
Реализована ролевая модель доступа: Ученик, Повар, Администратор.

## Функционал

### Ученик
- Авторизация (ID/Login/Email).
- Просмотр меню на текущий день.
- Онлайн-заказ питания.
- Оплата или подписка.
- Учет аллергенов.
- История заказов.

### Повар
- Выдача питания по идентификатору ученика.
- Управление меню (формирование на дату).
- Складской учет (мониторинг остатков, ручная корректировка).
- Управление номенклатурой (добавление блюд/продуктов).
- Создание заявок на закупку.
- Статистика выдачи за день.

### Администратор
- Статистика.
- Обработка заявок на закупку.
- Финансовые отчеты.
- Управление пользователями.

## Стек технологий
- **Backend:** Python 3.13, Flask
- **Database:** SQLite
- **Frontend:** HTML/CSS/JS
- **Deploy:** Docker, Docker Compose
